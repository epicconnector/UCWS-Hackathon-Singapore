# Contributing to UCWS Hackathon Singapore

Thank you for your interest in the UCWS Hackathon Singapore! This repository is the official platform for project collection, showcase, and documentation.

## Important: What This Repository Is

⚠️ **This is NOT an open-source community.** This platform exists to:
- Collect hackathon projects
- Showcase innovative AI solutions
- Preserve project documentation and resources
- Facilitate commercial partnerships and opportunities

---

## How to Participate

### 1. Register for the Hackathon

Visit our official registration page:
👉 [UCWS Hackathon Singapore - Registration](https://evol.epicconnector.ai/events/ucws-singapore-hackathon---2026-cxgy?invite=2geaJa5h)

### 2. Submit Your Project (Official Platform)

All official hackathon project submissions must be made through:
👉 [EPICConnector Platform - Project Submission](https://evol.epicconnector.ai)

### 3. Showcase Project (This Repository)

After official platform submission, you may optionally showcase your project here:

1. **Fork** this repository
2. **Create a new branch** (e.g., `projects/your-project-name`)
3. **Add your project folder** under `/projects/`
4. **Include comprehensive documentation** (see [SUBMISSION_GUIDELINES.md](docs/SUBMISSION_GUIDELINES.md))
5. **Commit with clear messages**
6. **Submit a Pull Request** with project details

---

## Submission Types & Review

### ✅ We Accept

- **Hackathon Project Documentation** — Code, demos, and team information from registered participants
- **Documentation Improvements** — Fixes to guides, FAQs, event information
- **Resource Suggestions** — Tools and learning materials relevant to hackathon participants
- **Bug Reports** — Issues with repository content or structure

### ❌ We Do NOT Accept

- **Open-source contributions** — This is not an open-source community
- **Unrelated projects** — Only UCWS Hackathon Singapore 2026 projects
- **Spam or promotional content** — Commercial solicitation without team connection
- **Code that violates our guidelines** — See CODE_OF_CONDUCT.md and SECURITY.md
- **Plagiarized or unauthorized content** — Must be your own work

---

## Guidelines for Project Submission

### Code of Conduct

All contributors and participants must adhere to our community values:
- **Innovation:** Push creative boundaries
- **Collaboration:** Support fellow builders and the community
- **Integrity:** Honest work and fair attribution
- **Diversity:** Welcome perspectives from all backgrounds
- **Respect:** Treat all community members with respect

### IP & Commercialization

By submitting to this repository:
- You **retain all intellectual property rights** to your project
- EPICConnector may **showcase and reference** your work for partnership opportunities
- Your project becomes eligible for **commercial partnerships and investment** through our network
- Your work is **archived for the long term** as part of the hackathon archive

See [LICENSE](LICENSE) for full terms.

### Quality Standards

Before submitting:
- [ ] Read [docs/SUBMISSION_GUIDELINES.md](docs/SUBMISSION_GUIDELINES.md)
- [ ] Follow proper folder and file structure
- [ ] Write comprehensive README documentation
- [ ] Include installation and setup instructions
- [ ] No sensitive information (API keys, passwords) in code
- [ ] Include .gitignore and proper configuration files
- [ ] Provide clear contact and team information
